## Basic I/O using `cin` and `cout` 
- `cout` and `<<`:
  - `cout << variable;`
- use `endl` or `\n` tp add line breaks
  - `cout << variable << endl;`
  - `cout << variable << "\n";`
- `cin` and `>>`:
  - `cin >> variable;`
- `cim ` can be chained
  - `cin >> var1 >> var2 >> var3;`
